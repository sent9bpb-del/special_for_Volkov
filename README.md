# Курсовая работа: матмодель реактора + оптимизация + имитационное моделирование со случайным входом

Этот репозиторий — законченный пайплайн курсовой по литовке:

1) **Математическая модель динамики** (интегрирование до стационара, концентрации C1–C4, выход HCHO).  
2) **Оптимизация параметров** \(T, V\) по максимизации HCHO (через минимизацию \(-HCHO\)). 
3) **Генерация случайного процесса** \(z(k)\) с заданными статистическими характеристиками. 
4) **Имитационное моделирование**: подача случайного входа \(m_{CH3OH,вх}(t)=z(t)\) → получение \(C1_{вх}(t)\) и \(C2_{вых}(t)\) (HCHO). 
5) **Проверочные прогоны** и визуализация результатов.

## Cтруктура репозитория

| Файл                        | Назначение                                                                      | Ключевые сущности / результат                                                                 |
| --------------------------- | ------------------------------------------------------------------------------- | --------------------------------------------------------------------------------------------- |
| `math_model_dynamics.py`    | Математическая модель динамики + целевые функции для оптимизации                | `solve_model(V,T,...)`, `hcho_objective(T,V)`, `pozitive_hcho_objective(T,V)`                 |
| `test_optimization.py`      | Метод покоординатного спуска + контурные графики + траектория поиска            | `coordinate_descent(...)`, `draw_contour_cd(...)`; прогон на тестовой функции и на матмодели  |
| `random_process.py`         | Генерация псевдослуч. последовательности и процесса `z(k)` с контролем M, σ², α | Итог: формируется `z` + графики `x(k)`, `z(k)`, `Kz(S)` и аппроксимация                       |

## Как запускать (рекомендуемый порядок)
### Шаг 1 — Проверить работоспособность матмодели

```bash
run math_model_dynamics.py
```

Что получаем:
Рабочую математическую модель

### Шаг 2 — Сгенерировать случайный процесс z(k)

```bash
run random_procces.py
```
Что получаем:
- последовательность x(k) и процесс z(k);
- оценка Mx, σx², Mz, σz², корреляции и параметра αz;
- если отклонение характеристик больше 10% — выполняется подбор параметров процесса.
